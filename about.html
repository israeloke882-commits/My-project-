About PhishGuard

Hybrid Cascading Machine Learning System

PhishGuard uses a sophisticated three-stage cascading approach to detect phishing emails with high accuracy and efficiency.

Our Methodology

Stage 1: Logistic Regression

The first line of defense performs fast lexical detection. It quickly identifies obvious phishing patterns using TF-IDF vectorization and linear classification. If confident (>70%), it makes an immediate decision.

Stage 2: Random Forest

When Stage 1 is uncertain, the Random Forest ensemble handles nonlinear patterns and ambiguous cases. Using 100 decision trees, it captures complex relationships in the data.

Stage 3: LSTM Neural Network

For the most challenging cases, our LSTM deep learning model provides contextual and semantic understanding. It analyzes email sequences to detect sophisticated phishing attempts.

Technology Stack

  • Backend: Flask (Python)
  • Machine Learning: scikit-learn, TensorFlow/Keras
  • NLP Processing: NLTK, TF-IDF Vectorization
  • Database: SQLite (for local development)
  • Authentication: Flask-Login

Key Features

  • Real-time phishing detection with confidence scores
  • Cascading pipeline for optimal accuracy and speed
  • Model transparency - see which stage made the decision
  • Analysis history tracking for registered users
  • Secure user authentication and data privacy
